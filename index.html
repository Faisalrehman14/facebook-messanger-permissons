📋 Live Permission Demos

Har tab ek permission ka use case demonstrate karta hai — App Review mein yahi dikhayen.

public_profile

Use case: User login & basic profile display

App user ki public profile (name, photo) dikhati hai taake unhe identify kiya ja sake.

Login karein taake profile load ho.
GET /me?fields=id,name,picture

pages_show_list

Use case: Page selection for messaging

User ko unki managed Facebook Pages ki list dikhati hai taake woh Page select kar sakein.

Loading...
GET /me/accounts?fields=id,name,category,picture,access_token

pages_read_engagement

Use case: Read Page posts & engagement metrics

Page ki posts, likes aur comments read karta hai taake engagement analytics dikhaye.

Pehle upar se Page select karein.
GET /{page-id}/posts?fields=message,created_time,likes.summary(true),comments.summary(true)

pages_messaging

Use case: Send & receive Page messages

Facebook Page se Messenger messages bhejna — customer support / chatbot ke liye.

POST /{page-id}/messages  { recipient: {id}, message: {text} }
PSID kaise milega? Test user ko Page par message karein → Webhooks setup karein → ya Graph API Explorer se /{page-id}/conversations check karein.

pages_utility_messaging

Use case: Order updates, account alerts (utility messages)

Transactional / utility messages — order status, appointment reminders, etc.

POST /{page-id}/messages  messaging_type: UTILITY
